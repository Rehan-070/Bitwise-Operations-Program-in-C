# 🔢 Bitwise Logical Operators Program in C

A simple C program that demonstrates the use of bitwise logical operators such as AND, OR, XOR, Complement, Left Shift, and Right Shift.

---

## 🚀 Features

- ➕ Bitwise AND (`&`)
- ➖ Bitwise OR (`|`)
- 🔁 Bitwise XOR (`^`)
- 🔄 Bitwise Complement (`~`)
- ⬅ Left Shift (`<<`)
- ➡ Right Shift (`>>`)
- 🧮 User input for two integers
- ⚡ Console-based execution

---

## 🧠 How It Works

1. User enters two integer values.
2. Program performs:
   - AND operation
   - OR operation
   - XOR operation
   - Complement operation
   - Left shift by 2 bits
   - Right shift by 2 bits
3. Results are printed on the console.

## 🛠️ Requirements

Make sure you have a **C compiler** installed.

For Windows:
- Install MinGW or use CodeBlocks

For Linux/Mac:
- GCC usually pre-installed

Check GCC version:

`gcc --version`

---

## ▶️ How to Run

# compile the file  
`gcc logical_oprators.c -o logical_oprators`

# run the program (Windows)  
`logical_oprators`

# run the program (Linux/Mac)  
`./logical_oprators`

---

## 🧪 Example Run

Enter the Three numbers : 5 3  

The logical AND of 5 and 3 is 1  
The logical OR of 5 and 3 is 7  
The logical XOR of 5 and 3 is 6  
Complement of given number is -6  
Left shift 20  
Right shift 1
